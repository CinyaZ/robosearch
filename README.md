# RoboSearch

RoboSearch is a staged course project for building an active visual search agent aimed at RoboCup@Home style object-finding tasks.

## Stage 0 Goal

Stage 0 only establishes:

- project structure
- shared data types
- config loading
- minimal demo entrypoint

At the end of Stage 0, the project should be able to print a minimal startup message through:

```bash
python -m demos.mock_search_demo
```

If `python` is not yet available on your machine, use the Windows Python launcher after installing Python:

```bash
py -m demos.mock_search_demo
```
